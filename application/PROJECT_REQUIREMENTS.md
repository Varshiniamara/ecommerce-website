# Project Requirements & Tech Stack Reference

## Pre-requisites
- **Node.js & npm**: Runtime environment and package manager.
- **React.js**: UI Library.
- **Vite** (Suggested in docs, but we are using Create-React-App): Build tool.

## Core Stack
- **Framework**: React.js
- **Styling**: Tailwind CSS
- **Routing**: React Router DOM
- **API Client**: Axios
- **Backend**: JSON-Server (Mock REST API)

## Requirements
- Full CRUD operations (Create, Read, Update, Delete).
- Responsive design.
- Navigation between pages.
